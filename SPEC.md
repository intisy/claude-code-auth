# Claude Code Auth - Specifications & Test Requirements

## Goal
Manage Claude Code models and subscription cycling with a unified UI.

## Requirements
- [ ] **UI Consistency**: Must share identical UI design and structure as antigravity-auth (share code where possible to ensure structural parity).
- [ ] **Cross-Platform Capability**: Allows Claude Code models to be utilized within OpenCode.
- [ ] **Subscription Cycling**: Automatically cycles through multiple active Claude Code subscriptions for both OC and CC.
- [ ] **OpenCode Integration**: Operates identically to the Antigravity plugin, but registers under the 'Anthropic' provider in OpenCode.
